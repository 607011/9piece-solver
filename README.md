# 9piece-solver

Solver for the 3x3 Scramble Puzzle, written in C++


## Prequisites

- C++11 compiler


## Build

```
git clone https://github.com/607011/9piece-solver.git
cd 9piece-solver
git submodule init
git submodule update --remote --merge
make
```


## Run

```
./solve data/puzzle-0000.txt
```
